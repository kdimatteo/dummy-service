Dummy Service
=============

Dummy service for testing data grids

* Retuns a collection whos length is indicated in the url (`localhost:3000/1`)
* Uses a default schema including `id`, `hash`, and `label` e.g.:
```js
[ 
  {
  "id": 0,
  "hash": "wgcmbk",
  "label": "banjo burrito banana"
  },
  { ... }
]
```
